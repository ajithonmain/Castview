Your screen, on any device with a browser

Mirror a Windows, Mac, or Linux screen to a phone, tablet, or another computer. One command on the laptop, a QR scan on the viewer. Nothing to install anywhere else.

$ npx castview
Needs Node.js on the shared computer — that's the only requirement.
01

Run one command

npx castview starts the server and opens a setup page with QR codes for every network you're on.

02

Scan with the camera

Point the phone or tablet camera at the QR code. The stream opens in its browser, already authenticated.

03

Watch live

Smooth cursor-included streaming over your local network. Nothing ever leaves it.

Built for the worst day of your laptop's life

Castview began as a rescue tool for a laptop with a dead display — so it assumes nothing works except Node and a cable.

Works without WiFi

A USB cable and phone tethering is enough. The setup page detects the connection and shows its QR automatically.

QR to connect

No addresses to type. Scan with the camera and the mirrored screen opens, PIN included.

PIN protected

A random PIN gates every session, so screen sharing on a shared network stays private.

Smooth streaming

Uses ffmpeg when available for fluid cursor-included capture, with a pure-JS fallback when it isn't.

Local only

Frames travel over your LAN and nowhere else. No accounts, no cloud, no telemetry.

No admin rights

Runs as a normal user. No installers, no drivers, no native builds — pure JavaScript dependencies.

The broken-screen story

A laptop display died. The machine still worked — there was just no way to see it. Castview turned a tablet into its screen using only a USB cable and one command typed blind. If that's the situation you're in right now: plug in your phone, enable USB tethering, run npx castview, and scan the code your phone camera finds on the terminal you can't see — it's printed to the setup page your phone can.
