USB cable — no WiFi needed

  1. Connect tablet/phone to this computer with a USB cable
  2. On the device, enable USB tethering (Settings → Network → Hotspot & tethering)
  3. Wait a few seconds — a new network adapter appears here automatically
  4. Restart Castview (Ctrl+C, then node server.js) and use the new address it prints, or refresh this page
Viewers on your network can watch this screen while the server runs. Stop it with Ctrl+C in the terminal when done. The stream stays inside your local network — nothing goes over the internet.
